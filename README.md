# ✨ RONY SKIES - WhatsApp Mini Bot

Bot completo com menus interativos, games, tools, downloads e PIX Premium!

## 🚀 Deploy no Railway
1. Fork este repo
2. Railway.app > New Project > GitHub
3. Add vars: `PIX_KEY=suachave` | `PIX_VALOR=9.99`

## 📱 Comandos
- `menu` - Menu principal
- `1` - Info
- `2` - Games
- `3` - Tools
- `4` - Downloads
- `5` - Support
- `6` - Premium PIX
